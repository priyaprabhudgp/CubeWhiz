# Rubiks-Cube-Solver
- Got the colour grid code from [Link](https://varunpatil.github.io/Rubiks-Cube-Solver/)
- This Solver uses the Kociemba algorithm to solve a Rubik's Cube efficiently

- Before running this program on your device, please install the TwoPhase Library and verify that server.js references the Python version you have
- We developed this on VS Code
